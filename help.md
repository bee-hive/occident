<h2>
How to Submit Your Image Collection
</h2>
<b>There are two ways to submit your image collection to the Occident database and web portal.<b>
<ul>
    <li>Use <a href="https://forms.gle/bSr2BMC8rJm1Yq278" target="_blank">our simple Google Form</a>. The form allows you to submit an image collection for a single trial and well. Reuse the form to submit multiple collections.</li>
    <li>Or, learn about <a href="https://github.com/bee-hive/occident/tree/main/submissions#readme" target="_blank">our tab-delimited file format</a> (TSV) and provide metadata on multiple trials and multiple wells (one per row) using a template of required column headings.</li>
</ul>
